Responsive multi-page website for **HLEGAL** law company. Built with Vite, SCSS, and vanilla JavaScript. Includes a mobile-first layout, multilingual support (UA / PL / EN), and pages for services, team, publications, and contact.

## Live demo

https://hlegalproject1.vercel.app/

## Pages

| Page | Path |
|------|------|
| Home | `index.html` |
| About us | `pages/about-us.html` |
| Services | `pages/services.html` |
| Team | `pages/team.html` |
| Publications | `pages/publications.html` |
| Contact | `pages/contact-page.html` |

## Features

- Responsive layout for mobile, tablet, and desktop
- Multilingual UI (Ukrainian, Polish, English)
- Interactive home tabs, services cards, team grid, and mobile navigation
- SCSS modules with BEM-style class naming
- Optimized images and SVGs at build time

## Tech stack

- Vite
- Sass (SCSS)
- JavaScript (ES modules)
- Vite
